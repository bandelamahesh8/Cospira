# COSPIRA TERMINOLOGY GOVERNANCE

> "One Concept. One Word."

This document serves as the single source of truth for all naming conventions in Cospira.
Inconsistencies confuse users. Discipline builds trust.

---

## 🏛️ CORE ENTITIES

| Concept               | Approved Term    | Banned Terms                                            | Rationale                                                                |
| :-------------------- | :--------------- | :------------------------------------------------------ | :----------------------------------------------------------------------- |
| The digital place     | **Room**         | Meeting, Session, Call, Channel                         | "Room" implies permanence and spatial presence. "Meeting" is temporary.  |
| The creator/owner     | **Host**         | Admin, Owner, Leader, Creator                           | "Host" implies hospitality and control, fitting for a social/work space. |
| The people connecting | **Participant**  | User, Member, Attendee, Guest (unless generic non-auth) | "Participant" implies active engagement. "User" is robotic.              |
| The instance of time  | **Session**      | Call, Meeting, Recording                                | A Room exists forever. A "Session" is the active time spent in it.       |
| The AI output         | **Intelligence** | Summary, Report, Notes, Transcript                      | "Intelligence" implies synthesized value, not just raw data.             |

---

## 🎮 CONTROLS & ACTIONS

| Concept                | Approved Term       | Banned Terms                      | Rationale                                                                           |
| :--------------------- | :------------------ | :-------------------------------- | :---------------------------------------------------------------------------------- |
| Ending the active time | **End Session**     | Close Room, Stop Meeting, Disband | You end the session (event), but the room (place) remains.                          |
| Deleting the place     | **Disband Room**    | Delete Room, Destroy, Close       | "Disband" implies a permanent breaking up of the group/space.                       |
| The main toolbar       | **Command Deck**    | Toolbar, Controls, Bottom Bar     | "Command Deck" aligns with the "Arena/Space" aesthetic.                             |
| Sharing screen         | **Present**         | Share Screen, Cast                | "Present" elevates the action. You are presenting content, not just sharing pixels. |
| The floating web view  | **Virtual Browser** | Browser, Web, Iframe              | "Virtual Browser" clarifies it's a cloud instance, not local.                       |

## 🎨 UI STATUS STATES

| State        | Approved Term                                                | Meaning |
| :----------- | :----------------------------------------------------------- | :------ |
| **Live**     | Participants are connected and media is flowing.             |
| **Upcoming** | Room is created but start time is in future.                 |
| **Paused**   | Session is halted (e.g. for break), but not ended.           |
| **Ended**    | Session is over, intelligence is generated. History remains. |
