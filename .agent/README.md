## 🎉 QUICK WINS

**Want to see it work?**

1. **Generate a summary**:

   ```bash
   # Join a room, chat for a bit, then:
   socket.emit('summary:generate', { roomId })
   ```

2. **See room timeline**:

   ```bash
   socket.emit('timeline:get', { roomId })
   ```

3. **Create an action**:

   ```bash
   socket.emit('action:create', {
     roomId, text: 'Test action', priority: 'high'
   })
   ```

4. **Get room stats**:
   ```bash
   socket.emit('timeline:get-stats', { roomId })
   ```
