03 — Depth-Scroll Tunnel
Between hero and features, a scroll-pinned section shows a 3D tunnel of glowing rings. User scrolls through at their own pace — each ring reveals a feature bullet as it passes the camera.
ScrollTrigger.create({
  pin: true, scrub: 1,
  onUpdate: (self) => {
    camera.position.z = lerp(80, 0, self.progress)
  }})
04 — Game Card Hologram
Game arcade cards use CSS 3D preserve-3d with a tilt that tracks mouse. On hover, a holographic shimmer shader sweeps across the face. Cards "pop" 24px toward viewer on focus.

05 — Pricing Morph
Pricing cards use WebGL plane geometry. Switching tiers morphs the card mesh via vertex displacement — numbers count up/down with spring physics. Background responds with a ripple.
