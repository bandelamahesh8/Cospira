Valid win patterns horizontal vertical diagonal (both directions) --- TURN RULES Players alternate turns. Player 1 → red pieces Player 2 → yellow pieces Example Player1 drops piece Player2 drops piece --- MOVE RULE Players select a column. The piece falls to the lowest empty cell. Example Column 3 selected grid before 0 0 0 1 2 2 new piece lands above last filled cell. --- INVALID MOVE RULES Move must be rejected if: column index < 0 column index > 6 column already full game status not active not the player's turn --- WIN RULES Win occurs when four consecutive pieces exist. Check patterns horizontal vertical diagonal ↘ diagonal ↙ Example horizontal 1 1 1 1 Example vertical 1 1 1 1 Example diagonal 1 1 1 1 --- DRAW RULE If board completely filled (42 moves) and no win occurs. Game status becomes draw. --- PHASE 5 — COSPIRA ROOM INTEGRATION Game entry path Room → Activities → Games → Connect4 Game panel opens inside room. Voice call remains active.
