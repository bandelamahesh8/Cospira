# COSPIRA MOBILE APP

## Project Scope

This project contains the mobile application for the **COSPIRA AI OS**, a next-level AI operating system featuring Control, Communication, Intelligence, and Simulation layers.

## Architecture

The folder structure follows a strict domain-driven design:

- **src/core**: Core logic, constants, theme.
- **src/screens**: Main application screens (Command, Rooms, AI Brain, Simulation, Profile).
- **src/components**: Reusable UI components.
- **src/services**: API integration.
- **src/store**: State management.

## Getting Started

1. Run `npm install`
2. Run `npm start` to launch Expo Go.
